# Senior Tech Hub Website

Senior Tech Hub is a Cape Town-based digital inclusion and online safety initiative helping seniors, families and first-time technology users build digital confidence.

This repository contains the website for Senior Tech Hub.

## Project Purpose

The website will help visitors understand:

* What Senior Tech Hub does
* Who we support
* Our digital safety and technology education services
* Tech Tips Africa content and learning support
* How to contact us for support, workshops or partnerships

## Main Website Sections

* Home
* About Senior Tech Hub
* How We Help
* Smartphone Basics
* Computer Skills
* Online Safety
* Tech Tips Africa
* Contact

## Target Audience

Senior Tech Hub supports:

* Seniors
* Parents and grandparents
* Families
* First-time smartphone users
* Community learners
* Women and underserved communities
* Organisations supporting digital inclusion

## Brand Style

The website should feel:

* Clear
* Trustworthy
* Warm
* Senior-friendly
* Professional
* Easy to read
* Mobile-friendly

Preferred colours:

* Navy blue
* White
* Light blue
* Orange accent

## Development Notes

Keep the website simple and accessible.

Use:

* Large readable text
* Strong contrast
* Simple navigation
* Clear buttons
* Mobile-friendly layout
* Fast loading pages

Avoid:

* Tiny text
* Crowded sections
* Complicated animations
* Confusing technical language
